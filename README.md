# Ibeju-Lekki LGA — Mockup (Klassrun Technologies)

Demo Next.js site showcasing a modern rebuild of the Ibeju-Lekki Local Government website.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy to Vercel

```bash
npx vercel
```

Or push to GitHub and import on vercel.com.

## Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Fraunces (display) + Inter (body) — via Google Fonts

## Pages

- `/` — Homepage (hero, executive chairman, news grid, stats)
- `/news` — Newsroom with featured + grid
- `/news/[slug]` — Article template

## Palette

- Forest `#0F3D2E` — primary
- Moss `#1B5E3F` — secondary
- Gold `#C89B3C` — accent
- Ink `#0A1F14` — text
- Bone `#FAF7F0` — background
